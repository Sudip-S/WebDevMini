CSS PRACTICE ASSIGNMENT

Css is a very important computer language that helps to style
and design the web site in a very attractive ways.

Css stands for Cascading Style Sheet which is used in HTML or XHTML to enhance the
style of a web site or a page.

It is a very important part of web development that helps the user or a developer to make the
web page very interesting and beautiful.

TYPES OF SELECTORS IN CSS

There are five types of selectors in CSS, They are:

  • Class Selector
  • ID Selector
  • Element Selector
  • Universal Selector
  • Group Selector

TYPES OF MAJOR DISPLAY PROPERTIES IN CSS

There are many display properties in CSS, Some of them are:

  1. Flex
  2. Inline
  3. Block
  4. Content
